Metagame speciális kártyák (ezekre épülnek)

Ezek a .card alapra épülő, már kész Metagame komponensek. Mindegyik a fenti alap stílusokat örökli (arany keret, sötét háttér) + saját egyedi layout/stílus.

Komponens CSS class Hol Ami egyedi
Termék kártya card mg-product-card Terméklista, hero ajánló Transzparens háttér, radial gradient kép, CTA félig kilóg, hover shadow
About — Bolt card mg-about-card--bolt Kezdőlap Függőleges layout, fix kép magasság
About — Klub card mg-about-card--klub Kezdőlap Vízszintes layout (flex-row), stat box-ok, nagyobb kép
About — Stat box card mg-about-stat Kezdőlap (klub kártyán belül) Mini kártya: szám + label, 2x2 grid
Contact — Fő kártya card mg-contact__card Kezdőlap Form + térkép egymás mellett (row + col-md-6)
Contact — Info kártya card mg-contact__info-card Kezdőlap (3 db: Bolt, Klub, Telefon) Ikon + szöveg + akció gomb, col-md-4 gridben
Related product card mg-related__card Termékoldal, cikk oldal sidebar Desktop column / mobil row, cover kép mobilon, radius-sm
Krónika — Kiemelt card mg-chronicle-card--featured Kezdőlap Cikk kép + tartalom + badge-ek, hover nélkül
USP feature card mg-usp-card Kezdőlap (6 db) Gradient háttér (nem bg-card), hex ikon, topológia SVG

Hogyan épül fel egy speciális kártya

A .card adja az alapot (keret, háttér, radius, flex-column). Az mg-* class hozzáadja az egyedi layoutot és stílust.

<!-- Új kártya Lacinak: standard Bootstrap -->
<div class="card">
  <div class="card-body">
    <h5 class="card-title">Cím</h5>
    <p class="card-text">Szöveg</p>
    <a class="btn btn-secondary">Gomb</a>
  </div>
</div>

<!-- Meglévő speciális kártya: card + egyedi class -->
<div class="card mg-product-card">
  <div class="mg-product-card__image">...</div>
  <div class="mg-product-card__info">...</div>
  <div class="mg-product-card__content">...</div>
</div>

Zöld = Bootstrap standard class-ok (dokumentáció: getbootstrap.com/docs/5.3/components/card)
Sárga = Metagame egyedi class-ok (csak specifikus komponenseknél)
